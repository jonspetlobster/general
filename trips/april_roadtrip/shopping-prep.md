# Shopping & Prep List

## 🚴 Mountain Bikes - Megan & Max

### Bike Options (New)

**Budget-Friendly ($500-800):**
- **Polygon Xtrada 5** (~$600) — great value hardtail, 1x drivetrain
- **Vitus Nucleus 27 VR** (~$550) — solid entry-level
- **Giant Talon** (~$700) — reliable brand, good components

**Mid-Range ($800-1500):**
- **Specialized Rockhopper Comp** (~$1,100) — excellent hardtail
- **Trek Marlin 8** (~$1,100) — versatile trail bike
- **Canyon Stoic 4** (~$1,300) — great geometry, solid components

**Full Suspension Entry (~$1500-2500):**
- **Giant Stance 29** (~$2,000) — if you want to go full squish

**Where to Buy:**
- **Mike's Bikes** (Santa Cruz/Bay Area) — local shop, test rides
- **REI** (Capitola) — returns are easy, membership dividend
- **Online:** Canyon, Polygon (direct-to-consumer pricing)

**Size Guide:**
- Measure inseam, use brand size charts
- Test ride if possible (especially for Megan)
- Moab = technical trails, go for sturdy over ultralight

### Bike Accessories (Per Person)

- [ ] Helmet (MIPS recommended)
- [ ] Gloves
- [ ] Padded shorts or bibs

- [ ] Bike multi-tool (carry on rides)

- [ ] Mini pump or CO2 inflator + cartridges
- [ ] Patch kit
- [ ] Chain lube

### Truck Bike Rack

- [ ] **Truck bed bike rack** or **tailgate pad**
  - **1Up USA Heavy Duty** (best, $$$$) — bomb-proof, easy load
  - **RockyMounts BackStage** (~$350) — swing-away, bed-mounted
  - **Tailgate Pad** (~$150-300) — cheap, works, bikes touch each other
  - **Thule Insta-Gater** (~$500) — premium tailgate solution

## Camping Gear (Fill Gaps)

### Need to Buy/Verify You Have

- [ ] **Sleeping bags rated for 30°F** (desert nights can drop to freezing)
- [ ] **Sleeping pads** (insulated for cold ground)
- [ ] **Tent large enough for 2 people + gear**
- [ ] **Camp stove + fuel** (MSR, Jetboil, or Coleman dual-fuel)
- [ ] **Cooler** (Yeti/RTIC if you want ice to last, Coleman works)
- [ ] **Water containers** (5-gallon jugs, collapsible, etc.) — need 10+ gallons total
- [ ] **Headlamps** (one per person + spare batteries)
- [ ] **Camp chairs** (if you don't have)
- [ ] **Portable toilet or WAG bags** (for dispersed camping)

## Vehicle Prep

### F150 Maintenance (Do Before Trip)

- [ ] **NEW TIRES - DEADLINE: March 22** (not last weekend before departure)
- [ ] Oil change (if due)
- [ ] Check tire pressure + tread on new tires (including spare)
- [ ] Inspect brake pads
- [ ] Top off all fluids (coolant, brake, washer, etc.)
- [ ] Test 4WD engagement (engage before trip to verify it works)
- [ ] Check battery health
- [ ] Replace windshield wipers if needed
- [ ] Wash/vacuum (start clean)

### Vehicle Gear to Buy/Check

- [ ] **Air compressor** (ARB, Viair, or cheaper option) — for airing down/up tires
- [ ] **Tire pressure gauge**
- [ ] **Recovery boards** (MaxTrax, ActionTrax) — optional but useful for sand/mud
- [ ] **Recovery strap + shackles** (if you don't have)
- [ ] **Spare fuel can** (5-10 gal, check permit rules for White Rim)
- [ ] **Fire extinguisher** (ABC rated, mount in cab)
- [ ] **First aid kit** (vehicle + camping)
- [ ] **Jumper cables or jump starter pack**
- [ ] **Road flares or reflective triangles**
- [ ] **Shovel** (collapsible or full-size)

## 🚁 Drone

**Options:**
- **DJI Mini 4 Pro** (~$759) — Best value, <249g (no FAA registration), 34 min flight time, 4K/60fps
- **DJI Air 3** (~$1,099) — Dual cameras, 46 min flight time, obstacle avoidance
- **DJI Mavic 3** (~$2,199) — Pro-level, Hasselblad camera, 43 min flight time

**Accessories:**
- [ ] Extra batteries (2-3 additional)
- [ ] ND filters for desert sun
- [ ] Hard case for transport
- [ ] Car charger
- [ ] MicroSD cards (128GB+ high-speed)

**Note:** Check NPS regulations — drones prohibited in National Parks (Canyonlands, Arches, Zion, Death Valley, Capitol Reef). OK on BLM land and most dispersed camping areas.
